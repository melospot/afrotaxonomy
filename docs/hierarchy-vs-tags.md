# Hierarchy Vs Tags

## Hierarchy
- `parents` defines lineage (what a genre is a type of).
- Use hierarchy for durable structural relationships.

## Tags
- `tags` adds searchable descriptors (language, mood, rhythm, geography).
- Use tags for context and discovery, not lineage.

## Rule of thumb
- If removing the link changes genre identity, use hierarchy.
- If removing the link only reduces discoverability, use tags.
